# Coding Standards

## Naming Conventions

- **File naming**: PascalCase.c/.h for modules
- **Function naming**: Module_Action()
- **Macro naming**: EPOCHSYNC_MAX_SERVERS
- **Enum naming**: NetworkStatus
- **Struct naming**: NetworkConfig
- **Header guard naming**: EPOCHSYNC_<MODULE>_H to avoid collisions and ensure consistency as codebase grows
